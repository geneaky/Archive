
리액티브 프로그램의 표준 API 사양을 말한다.
`비동기 데이터 스트림`과 `논-블로킹 백프레셔`에 대한 사양을 제공한다.

리액티브 스트림은 TCK(Technology Compatibility Kit)을 지원하기 때문에 라이브러리가 정해진 사양에 맞게 구현되었는지 보장할 수 있다.
